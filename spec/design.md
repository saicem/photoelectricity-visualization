# UI 设计规范

## 设计风格

科技感深色主题，灵感来源于光学实验室环境和光通信设备界面。

### 配色

| 色名 | 色值 | 用途 |
|------|------|------|
| 背景色 | `#0a0e17` | 页面主背景 |
| 卡片背景 | `#1a2332` | 面板/卡片 |
| 激光红 | `#ff3366` | 高能光波 |
| 激光绿 | `#00ff88` | 信号指示 |
| 激光蓝 | `#3366ff` | 中性光波 |
| 激光紫 | `#a855f7` | Q 路/调制器 |
| 激光橙 | `#ff8800` | 警告/相位信息 |
| 文字主色 | `#ffffff` | 标题/强调 |
| 文字次要 | `#94a3b8` | 正文说明 |
| 边框 | `#334155` | 分隔线 |

### 字体

- **标题**：`"Orbitron", "Rajdhani", sans-serif`
- **正文**：`"Inter", "Noto Sans SC", sans-serif`
- **代码/数据**：`"JetBrains Mono", monospace`

### 组件风格

- 圆角按钮 (`border-radius: 8px`)，悬停发光效果
- 卡片式布局，大量留白
- 左侧固定导航 + 右侧内容区
- 网格系统布局

### 装饰元素

- 背景网格线模拟光学平台
- 渐变光晕效果模拟光源
- 虚线模拟光路
